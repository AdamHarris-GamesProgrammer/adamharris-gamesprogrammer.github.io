About

Introduction

Belghast's Dominion is a ongoing project in which I am creating a full-fledged third person fantasy RPG with multiple enemy types, quests, dialogues and important choices to make. Belghast's DOminion is my first time creating my own particle systems, and using animations from Synty Studios. Currently in the game is Combat and the enemy AI systems. Enemies and the player can also level up as they fight. I hope to finish this project by the end of 2021

Features Remaining

  • Quests
  • Dialogue
  • Full-Size Maps
  • Inventory

Technologies Used

C#

Belghast's Dominion is written in C# using Unity 2019.4 this was chosen as the pre existing engine allows me to focus on more important aspects of the project such as gameplay. It also makes it alot easier as opposed to pure OpenGL or DirectX code.

Source Control using GitHub

I used GitHub to act as my source control system for this project, this project was transferred from home and to university a lot meaning that I had to make use of source control to keep a singular version between devices.
